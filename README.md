# README v0.3 — Марафон 2.0

Программа саморазвития для участников и наставников. Репозиторий разделён на два слоя: метамодель хранит основания и конструкторы, а модели применения содержат готовые шаги для работы.

## Два слоя материалов
- **Метаматериалы (мета-модель)** — описания состояний, методов, шаблоны и инструменты для разработки методичек.
- **Материалы применения (модели применения)** — линейные сценарии встреч и опросники, которые отдаются участникам и наставникам.
